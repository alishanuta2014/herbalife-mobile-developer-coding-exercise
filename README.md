# Herbalife Nutrition Tracker: Coding Exercise

A React Native starter app for a take-home coding exercise.

## Quick Start

```bash
npm install
npx expo start
```

- Press `i` for iOS Simulator
- Press `a` for Android Emulator
- Scan the QR code with [Expo Go](https://expo.dev/client) on a physical device (iOS or Android)

## Full Instructions

See [EXERCISE.md](./EXERCISE.md) for the exercise spec, [TODO.md](./TODO.md) for the prioritized feature list, and [AI-USAGE.md](./AI-USAGE.md) for the AI usage log you'll fill in.

## Quality Checks

```bash
npm run lint        # ESLint
npm run typecheck   # tsc --noEmit
npm run format      # Prettier
npm run verify      # all of the above (run before submitting)
```

Please keep `lint` and `typecheck` green in what you submit.

## Tech Stack

- React Native + Expo (TypeScript)
- React Navigation (Bottom Tabs)
- `@expo/vector-icons` (Ionicons)

## Requirements

- Node.js 18+
- One of the following to run the app:
  - **iOS Simulator:** requires Xcode on macOS
  - **Android Emulator:** requires Android Studio
  - **Physical device:** install [Expo Go](https://expo.dev/client) (iOS or Android)
